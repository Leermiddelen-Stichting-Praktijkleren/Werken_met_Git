#  Git Oefening - Stichting Praktijkleren

Welkom bij deze oefen-repository. Dit project wordt gebruikt om de basisprincipes van versiebeheer met Git en GitHub te leren.

![Stichting Praktijkleren](https://www.stichtingpraktijkleren.nl/typo3conf/ext/spl_sitepackage/Resources/Public/Images/SPL-logo.svg)


##  Inhoud van de repository

Deze repository bevat een eenvoudige webpagina structuur:
* **test.html**: De hoofdpagina van het project.
* **.gitignore**: Configuratie om systeembestanden en IDE-instellingen te negeren.
* **README.md**: Dit bestand met uitleg.

##  Aan de slag

Volg de instructies van je docent of het lesmateriaal om met deze repository te werken. De basisstappen zijn vaak:

1.  **Clone** deze repository naar je lokale machine:
    git clone <jouw-repo-url>
2.  Open de map in je favoriete editor (bijv. VS Code of PhpStorm).
3.  Open `test.html` in je browser om het resultaat te bekijken.
